# API de Seguros

Proyecto backend en **Spring Boot** para la gestión de **usuarios** y **pólizas de seguros**.

## 🚀 Tecnologías

- **Java 17+**
- **Spring Boot 3.x**
- **Spring Data JPA**
- **Hibernate**
- **Swagger / OpenAPI 3**
- **H2 Database** (para desarrollo)
- **Maven**
